# SportTech - Premium Company Profile Website

A modern, premium company profile website built with vanilla HTML, CSS, and JavaScript. Inspired by top-tier startup landing pages with clean design, smooth animations, and exceptional user experience.

## ✨ Premium Features

### 🎨 Design Improvements

- **Refined Color System**: Premium cyan-to-purple gradient with carefully selected colors
- **Better Typography**: Complete type scale with proper hierarchy and letter spacing
- **Enhanced Spacing**: Premium spacing system from 4px to 128px for perfect proportions
- **Subtle Gradients**: Background mesh gradients and glassmorphism effects
- **Premium Shadows**: Multi-layered box shadows for depth and dimension
- **Smooth Transitions**: Cubic-bezier easing functions for buttery smooth animations

### 📱 Mobile Experience

- **Mobile-First Approach**: Optimized for all screen sizes (360px to 4K)
- **Responsive Breakpoints**: 1024px, 992px, 768px, 480px, 360px
- **Enhanced Mobile Navigation**: Backdrop blur with smooth slide animation
- **Touch-Optimized**: Better tap targets and touch interactions
- **Stacked Layouts**: Smart content reflow for small screens

### 🚀 Interactive Features

- **Scroll Progress Bar**: Visual indicator at the top of the page
- **Premium Button Ripple**: Material Design-inspired click effects
- **3D Card Hover**: Subtle parallax tilt effect on service cards
- **Smooth Counter Animation**: Eased counting with stagger effect

- **Passive Event Listeners**: Better scroll performance
- **IntersectionObserver**: Efficient viewport detection
- **Reduced Motion Support**: Respects user accessibility preferences
- **Optimized Animations**: requestAnimationFrame for smooth 60fps
- **Lazy Loading Ready**: Infrastructure for image optimization

## 🎨 Customization

### Colors
Edit CSS variables in `style.css`:
```css
:root {
    --primary-color: #06b6d4;
    --secondary-color: #8b5cf6;
    /* ... more colors */
}
```

### Typography
Adjust font sizes and weights:
```css
:root {
    --font-size-base: 1rem;
    --font-weight-bold: 700;
    /* ... more typography */
}
```

### Spacing
Modify the spacing scale:
```css
:root {
    --spacing-md: 1rem;
    --spacing-lg: 1.5rem;
    /* ... more spacing */
}
```
